# 🐱🐶 Cats vs Dogs Image Classification using CNN

## 📌 Project Overview

This project focuses on building a Deep Learning-based Image Classification System capable of distinguishing between cats and dogs using Convolutional Neural Networks (CNNs). The model is developed using TensorFlow and Keras and trained on the Kaggle Cats vs Dogs dataset. The project demonstrates the complete workflow of a Computer Vision application, including data collection, preprocessing, model building, training, evaluation, and prediction on unseen images.

Image classification is one of the most important applications of Artificial Intelligence and Deep Learning. By using CNNs, the model automatically learns visual features such as edges, textures, shapes, and patterns from images and uses these learned features to classify whether an image belongs to a cat or a dog.

---

# 🎯 Objectives

The primary objectives of this project are:

- To understand the fundamentals of Computer Vision.
- To implement a Convolutional Neural Network (CNN) for binary image classification.
- To learn image preprocessing techniques.
- To apply Batch Normalization and Dropout for better model performance.
- To evaluate model accuracy and loss during training.
- To classify unseen images of cats and dogs.
- To gain practical experience with TensorFlow and Keras.
